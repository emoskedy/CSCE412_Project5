Projects that I have made

Developed a 2-player tic-tac-toe game

In my first year at Texas A&M University, I had the opportunity to work on a final project with my two other friends. Each of us had to make a game using Python then combine them into one file so the user can choose one of the three games. First, each of us created a hierarchy for the design of our program using a bottom-up approach. Then, we started working on implementing the design. Finally, we combined our codes into one single file and tested the program to make sure that it worked as intended; the program will ask the user to enter the game that they want to play or stop to quit the program. I made a game of tic-tac-toe on a 3x3 board and the board will be displayed before every move.

I chose tic-tac-toe because it is basic for someone who is new to programming and also to practice my skills. During the project, I learned how to plot from matplotlib, numpy on my own and also learned different methods to make the code more efficient.
